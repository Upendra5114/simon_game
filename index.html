Rules:
Gameplay:
1.The game presents a sequence of colors that flash in a random order.
2.The player must repeat the sequence by clicking the buttons (colors) in the correct order.
3.After each correct repetition, the sequence gets longer by one color.
Game Over: If the player clicks the wrong button or fails to repeat the sequence correctly, the game ends.
Winning: The game continues until the player completes a specific number of levels or until they make an error, causing the game to end.
